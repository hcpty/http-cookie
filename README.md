A note about HTTP Cookie.

### HTTP Cookie

HTTP Cookie的一个缺点是Client每一次发送HTTP请求的时候都只能发送HTTP Cookie的全部字段，而不是本次HTTP调用和服务所涉及的那些字段，导致这些HTTP请求的语义含糊。
